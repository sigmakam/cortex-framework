# Layout Instructions Examples

Use these as inspiration for the `--file` flag of the theme generator.

---

## Example 1: SaaS Landing Page

Modern SaaS landing page with a clean, professional look. Think Linear or Vercel.

- Primary color: deep indigo (#4F46E5)
- Secondary color: cool slate (#475569)
- Accent color: vibrant emerald for CTAs (#10B981)
- Background: pure white with subtle gray sections
- Typography: Inter for body, JetBrains Mono for code
- Border radius: medium (0.5rem on cards, 0.375rem on buttons)
- Shadows: subtle, layered shadows for depth
- Hero section: gradient background from indigo to purple, white text, centered headline
- Navigation: sticky top bar with logo left, links center, CTA button right

---

## Example 2: Business / Corporate Website

Trustworthy, established business presence with classic proportions.

- Primary color: navy blue (#1E3A5F)
- Secondary color: warm gold (#D4A847)
- Accent color: confident red for important actions (#DC2626)
- Background: off-white (#FAFAF8)
- Text color: near-black (#1A1A1A)
- Typography: Georgia or serif stack for headings, system sans-serif for body
- Border radius: small (0.25rem) for a crisp, professional feel
- Minimal shadows, rely on borders and spacing
- Wide container with generous padding

---

## Example 3: Creative Portfolio

Bold, expressive portfolio for a designer or creative agency.

- Primary color: hot pink (#EC4899)
- Secondary color: electric purple (#8B5CF6)
- Accent color: bright yellow (#FBBF24)
- Background: dark charcoal (#18181B)
- Text color: white (#FFFFFF)
- Muted text: warm gray (#A1A1AA)
- Typography: Space Grotesk or a geometric sans-serif
- Border radius: large (1rem) for friendly, rounded cards
- Bold shadows with colored tints
- Full-bleed images and asymmetric layouts

## Design Preferences Section

You can also describe preferences in plain language:

"I want a warm, inviting design. Earth tones like terracotta and olive green.
Rounded corners everywhere. Large, readable text. Lots of whitespace.
The font should feel handcrafted -- maybe a humanist sans-serif like Nunito.
CTAs in a warm coral color that stands out against the cream background."
